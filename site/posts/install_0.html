Your first action to create a TypexJS site is to fork typexjs repository inside the directory where you want to develop your new site on your local computer , then create a Github repository with the same directory name for instance, the Github repository for TypexJS is typexjs. .

Install Eleventy and the required Node packages locally, as they appear in your package.json file
